# 가계부 웹앱 제작 강의 정리

## 1. 플랜 모드로 요청
플랜 모드에서 이렇게 요구했다.

> "가계부 웹앱을 만들고 싶어. 어떤 구조·내용·기능이 필요할지 계획을 세워"

## 2. Claude가 선택을 요청함
Claude는 세 가지를 선택하라고 했다.

1. **기술 스택** — 순수 HTML/CSS/JS / React+Vite / Next.js+DB
2. **저장 방식** — 브라우저 localStorage / JSON 파일 백업 / 클라우드 DB
3. **기능 범위** — 기본(수입·지출 입력·목록·잔액) / 카테고리 분류 / 통계 차트 / 월별 예산 관리

## 3. 나의 선택
세 가지 모두 답했다.

1. **기술 스택** → 순수 HTML/CSS/JS (추천) 선택
2. **저장 방식** → 브라우저 localStorage (추천) 선택
3. **기능 범위** → "모두 다 포함되었으면 좋겠네" (전체 기능 요청)

## 4. 추가 요청 사항
1. **한 화면 통합** — 내역·통계·예산·설정을 탭 없이 한 화면에 모두 보이도록 수정 (사용 빈도에 따라 영역 크기 결정)
2. **스타일 참고** — https://refero.design/ 에서 원하는 스타일을 골라 md로 추가한 뒤, 그 스타일대로 UI 수정 요청
